# Music Server

A simple server for streaming and managing music collections.

## Features

- Stream music files over HTTP
- Manage playlists
- Search and browse library

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/tony-inventor/music-server.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the server:
   ```bash
   npm start
   ```
